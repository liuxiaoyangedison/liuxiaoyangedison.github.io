API接口规范

API 使用RESTful格式。
API主要用于甲方案件导入。

返回码

状态码 描述
1000 success
403 没有权限
501 服务器错误
401 参数无效
406 查找不到数据

返回格式

产品

添加产品

  1. url

    http://api.yicuicui.com/v1/product/add

  2. 格式

    json

  3. http请求方式

    post

  4. 输入参数

    参数名称 | 是否必须 | 类型 | 描述
    —– | —– | —– | —–
    companyName | 是 | String | 机构名称（公司名称）
    productName | 否 | String | 产品名称
    groupId| 是 | int | 产品所属组Id
    settleCyle| 否 | int | 结算周期 1: 按月; 2: 按季；默认按月
    formula| 是 | String | 佣金计算公式

    说明：

    • 非必填字段为更新的字段，根据需要填写

  5. 请求示例

  6. 返回参数

    1. 返回参数说明

      参数名称 | 类型 | 描述
      —– | —– | —–
      id| int | 产品 id

    2. 正确返回示例

3. 错误返回示例

更新产品

  1. url

    http://api.yicuicui.com/v1/product/update

  2. 格式

    json

  3. http请求方式

    post

  4. 输入参数

    参数名称 | 是否必须 | 类型 | 描述
    —– | —– | —– | —–
    id| 是 | int | 产品id
    companyName| 是 | String | 机构名称（公司名称）
    productName| 否 | String | 产品名称
    settleCyle| 否 | int | 结算周期 1: 按月; 2: 按季；默认按月
    formula| 是 | String | 佣金计算公式

    说明：

    • 根据传入的产品id更新产品
    • 非必填字段为更新的字段，根据需要填写

  5. 请求示例

  6. 返回参数

    1. 返回参数说明

      参数名称 | 类型 | 描述
      —– | —– | —–
      id| int | 产品 id
